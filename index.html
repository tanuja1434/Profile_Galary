Responsive design adapts layout to different screen sizes (CSS Grid/Flex + media queries). This page uses Grid to reflow cards from many columns on desktop down to a single column on phones.

Progressive enhancement starts with a basic, accessible experience, then layers on JS features. Here, the <noscript> fallback shows photos even if JS is off. With JS enabled, we enhance with dynamic rendering (loops), conditionals (favorites filter), a theme toggle, and a search filter.
